Completeness


Completeness is a concept we normally associate with, say, wavelets rather than with rank features. An image representation is complete if it encodes enough information to allow one to reconstruct the original image.

Clearly, reconstruction is not possible in the case of rank features, since the pixel intensity values are discarded after the ranking, and only rank information is preserved. However, we can redefine our notion of completeness as follows:

Definition: A set of rank features is complete if knowledge of the features uniquely specifies the ranking of the pixels in the original image.

Note that this requirement is
  • maximal: the ranking of the original image is all the information than rank features can possibly encode;
  • non-trivial: rank features are normally computed by sorting local image neighbouhoods, and it is in general not clear how the local rankings combine to allow comparisons between the intensity of distant pixels.
It can be proved that ranklets constitute a complete set of rank features.

The image to the left represents a complete set of ranklets for a one-dimensional, eight-pixel image. The notation is in analogy with Haar wavelets: the upper part of each step represents the T set, with the lower part corresponding to the C set.
